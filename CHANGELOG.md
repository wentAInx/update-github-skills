# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-06

### Added

- Initial `update-github-skills` skill for Codex, Claude Code, and other agent skill folders.
- Git-backed and manifest-backed skill inventory, dry-run checks, updates, and backup support.
- Cross-platform installers: `install.sh` for macOS/Linux and `install.ps1` for Windows PowerShell.
- Codex plugin marketplace manifest in `.codex-plugin/plugin.json`.
- Chinese and English README documentation.
